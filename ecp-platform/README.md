# ECP application workspace

This directory contains the runnable Next.js application for the Enterprise Car Configuration Platform.

## Use the repository-level docs first

The primary delivery documentation lives at the repository root:

- `../README.md`
- `../SETUP.md`
- `../USER_GUIDE.md`
- `../TESTING.md`
- `../ARCHITECTURE.md`

## Common commands

Run these from this directory:

- `npm run dev`
- `npm run lint`
- `npm run test`
- `npm run build`
