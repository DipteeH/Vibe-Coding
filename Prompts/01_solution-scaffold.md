# Prompt 01 - Solution Scaffold

## Intent

Design a production-quality enterprise car configuration platform that is locally runnable, professionally structured, and suitable for demonstration.

## Working prompt

Build a complete full-stack Enterprise Car Configuration Platform with a strong professional architecture. Use a modern TypeScript stack, implement both frontend and backend, make it locally runnable, and include documentation, validation, and testing. Favor a structure that keeps business rules separate from UI and persistence.

## Resulting direction

- Next.js App Router for UI and API delivery
- TypeScript throughout
- Prisma + SQLite for local persistence
- layered architecture across domain, application, infrastructure, and presentation
