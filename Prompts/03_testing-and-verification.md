# Prompt 03 - Testing and Verification

## Intent

Ensure the delivered application is verifiable and safe to demonstrate locally.

## Working prompt

Add unit, integration, UI, and end-to-end tests. Validate API routes, service logic, UI progression, and the save flow. Include professional delivery documents and capture any environment limitations that affect browser automation.

## Resulting direction

- Vitest-based unit, integration, and component coverage
- Playwright smoke test source for browser automation
- real-browser runtime verification of the save flow
- explicit documentation of local TLS limitations impacting Playwright downloads
